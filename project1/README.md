PriceTracker — Responsive Frontend Interface

DecodeLabs Full Stack Internship — Project 1 (Batch 2026)

A responsive, mobile-first web page that compares a product's price across Amazon, Flipkart, Meesho, Myntra, and a brand's own store — built with plain HTML, CSS, and JavaScript, no frameworks.


Overview

PriceTracker lets a shopper search for a product once and see it lined up side by side across five stores, cheapest first, with the savings called out. This project covers the interface phase of the internship: a flexible, accessible frontend built before any backend is wired in.
